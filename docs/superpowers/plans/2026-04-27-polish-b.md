## Task 1: 読み札アニメーションを修正する（CSS）

**Files:**
- Modify: `src/styles/app.css`

- [ ] **Step 1: `.reading-char` の初期状態とアニメーション設定を変更する**

`src/styles/app.css` の `.reading-char` ブロックを以下に書き換える。

```css
.reading-char {
  display: inline-block;
  opacity: 0;
  filter: blur(8px);
  animation: reveal-reading-char 1.2s ease-out both;
}
```

変更点:
- `opacity: 0.08` → `opacity: 0`（遅延中も完全に非表示）
- `filter: blur(10px)` → `filter: blur(8px)`
- `0.65s ease-out forwards` → `1.2s ease-out both`（`both` で遅延中も 0% キーフレームを維持）

- [ ] **Step 2: `@keyframes reveal-reading-char` を更新する**

同ファイルの `@keyframes reveal-reading-char` ブロックを以下に書き換える。

```css
@keyframes reveal-reading-char {
  0% {
    opacity: 0;
    filter: blur(8px);
    transform: translateY(4px);
  }

  60% {
    opacity: 0.6;
    filter: blur(3px);
  }

  100% {
    opacity: 1;
    filter: blur(0);
    transform: translateY(0);
  }
}
```

変更点:
- 0%: `opacity: 0.08` → `opacity: 0`
- 0%: `blur(10px)` → `blur(8px)`
- 0%: `translateY(10px)` → `translateY(4px)`（縦書きモードでの視覚干渉を軽減）
- 60%: `opacity: 0.55` → `opacity: 0.6`
- 60%: `blur(5px)` → `blur(3px)`

Expected: エラーなし。`dist/` が生成される。
